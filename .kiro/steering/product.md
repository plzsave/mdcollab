# Product Overview

mdcollab は Markdown ドキュメントの共同編集・レビュー Web アプリ。Google Apps Script 製の前身 `md-collab` の**脱 GAS 後継**で、ドキュメント・コメント・AI レビューを自前バックエンドで保持する。小規模で相互に信頼できるワークスペース（チーム / 個人）での、設計書・仕様書・テスト項目などの作成とレビュー運用を主対象とする。

## Core Capabilities

- **Markdown 編集 / プレビュー** とドキュメント整理（フォルダ＝タブ、ステータス、担当者割り当て、アーカイブ）
- **インラインコメント** — 選択範囲へのコメント、`@表示名` メンション、返信・解決・再開、アプリ内通知
- **AI レビュー（エージェント化）** — anthropic / openai を選択。tool use ループで参照リポジトリ・関連文書・既存コメントスレッドを**自分で読んで根拠付きでレビュー**する
- **AI 修正反映** — 指摘を反映した下書きを生成し、確認のうえ本文へ
- **メンバー管理と権限制御**（owner / member）

## Target Use Cases

- チームで md の設計・仕様・テスト項目をレビューする運用（合否・担当の可視化を含む）
- AI に「実コード／関連文書を踏まえた」レビューをさせ、人間が承認して反映する

## Value Proposition

- **ポータブル一本化**: Web 標準コアを1本書き、Cloudflare / AWS / GCP のいずれにもデプロイできる（差はアダプタ層だけ）。ベンダーロックを最小化し、個人＝CF / 職場＝AWS のような二面運用を1コードで吸収する。
- **データを自前で所有**: 本体・メタ・AI キーを自前ストア / DB に保持（GAS + Drive + Sheets の二重真実を解消）。
- **根拠付き AI レビュー**: 単発プロンプトでなく、ツールで文脈を自分で集めてレビューするエージェント方式。
